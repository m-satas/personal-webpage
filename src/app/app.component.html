About

Fast-forward to today I have a computer science university degree in a program focused on coding and visual design. My most relevant and recent experience is of a full stack web developer at Avilda developing Avilda's own warehouse management system with Angular, Node.js, Express, MySQL stack.

However I've been coding for longer than that and at (almost) every place I've worked. Whether it was coding algorithms for fully automated warehouse simulations using C# or for data processing as a data analyst with Python (and some VBA).

Now I continue to explore coding and visual design, driven by the joy of creation and the lessons learned from the process.
